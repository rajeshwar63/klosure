# Step 01 — Today's focus collapses to a one-line headline

**Sprint:** A (Dashboard)
**Goal:** The dashboard banner shows a single sentence by default. Tap to expand into the full Klo paragraph.

## The problem today

The "Today's focus" banner currently shows a 5-sentence paragraph on every dashboard load. Sales people open the dashboard 15-30 times a day. They don't need an essay every time — they need the one thing that matters most.

## The new behavior

**Default state — collapsed.** Shows:

```
◆ Klo · today's focus · today, 2:23 PM
DIB — send proposal today before Monday's demo.       [v]
```

One sentence. Generated by Klo (already happens — first sentence of the paragraph is the headline). Tap the chevron `[v]` or anywhere on the row to expand.

**Expanded state.** Shows the full paragraph as it does today, with `[^]` chevron to collapse.

**First-of-day rule.** The banner is expanded automatically on the first dashboard load each day. Once the user collapses it (or scrolls past — see "auto-collapse" below), it stays collapsed for the rest of the day.

**Auto-collapse on scroll.** If the user scrolls down past the banner without collapsing, collapse it automatically. They've seen it; don't keep showing the wall.

## Files touched

- `src/components/DailyFocusBanner.jsx` — main change
- `src/services/dailyFocus.js` — small helper to extract the headline sentence

## Headline extraction

Take the first sentence (split on `. ` then take index 0). If it's longer than 80 characters, truncate at the nearest word boundary and append `…`.

```javascript
// services/dailyFocus.js
export function extractHeadline(focusText) {
  if (!focusText) return '';
  // Strip markdown bold/asterisks for the headline
  const clean = focusText.replace(/\*\*/g, '').trim();
  const firstSentence = clean.split(/(?<=[.!?])\s+/)[0] ?? clean;
  if (firstSentence.length <= 80) return firstSentence;
  // Truncate at last space before 80 chars
  const truncated = firstSentence.slice(0, 80);
  const lastSpace = truncated.lastIndexOf(' ');
  return (lastSpace > 0 ? truncated.slice(0, lastSpace) : truncated) + '…';
}
```

## Component changes

```jsx
// DailyFocusBanner.jsx
import { useState, useEffect, useRef } from 'react';
import { fetchDailyFocus, extractHeadline } from '../services/dailyFocus';

const STORAGE_KEY = 'klosure:focusBanner:lastCollapsedDate';

function todayKey() {
  return new Date().toISOString().slice(0, 10); // 'YYYY-MM-DD'
}

function shouldStartExpanded() {
  const lastCollapsed = localStorage.getItem(STORAGE_KEY);
  return lastCollapsed !== todayKey();
}

export default function DailyFocusBanner() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [refreshing, setRefreshing] = useState(false);
  const [expanded, setExpanded] = useState(shouldStartExpanded());
  const bannerRef = useRef(null);

  // existing load() and useEffect for fetching focus stay the same

  // Auto-collapse on scroll
  useEffect(() => {
    if (!expanded) return;
    function onScroll() {
      if (!bannerRef.current) return;
      const rect = bannerRef.current.getBoundingClientRect();
      if (rect.bottom < 0) {
        // Banner is fully scrolled past — collapse and remember
        collapse();
      }
    }
    window.addEventListener('scroll', onScroll, { passive: true });
    return () => window.removeEventListener('scroll', onScroll);
  }, [expanded]);

  function collapse() {
    setExpanded(false);
    localStorage.setItem(STORAGE_KEY, todayKey());
  }

  function expand() {
    setExpanded(true);
  }

  function toggle() {
    expanded ? collapse() : expand();
  }

  if (loading && !data) return <div className="daily-focus-banner skeleton" />;
  if (!data || !data.focus_text) return null;

  const headline = extractHeadline(data.focus_text);

  return (
    <div ref={bannerRef} className={`daily-focus-banner ${expanded ? 'is-expanded' : 'is-collapsed'}`}>
      <button className="daily-focus-toggle" onClick={toggle} aria-expanded={expanded}>
        <span className="daily-focus-tag">◆ Klo · today's focus</span>
        <span className="daily-focus-when">{formatWhen(data.generated_at)}</span>
        <span className="daily-focus-chevron">{expanded ? '⌃' : '⌄'}</span>
      </button>

      {expanded ? (
        <>
          <div className="daily-focus-text">{data.focus_text}</div>
          <button className="daily-focus-refresh" onClick={() => load(true)} disabled={refreshing}>
            {refreshing ? 'refreshing…' : 'refresh'}
          </button>
        </>
      ) : (
        <button className="daily-focus-headline" onClick={expand}>
          {headline}
        </button>
      )}
    </div>
  );
}
```

## Visual treatment

- **Collapsed:** Small (~48px tall total), single line of text. Same blue tint background as today, but compressed padding (8px vertical instead of 16px).
- **Expanded:** Same as today's full banner, with the toggle chevron in the top-right.
- **Transition:** A simple `max-height` CSS transition for the expand/collapse, ~200ms ease-out. No bounce or fancy easing.

```css
.daily-focus-banner {
  transition: max-height 0.2s ease-out;
  overflow: hidden;
}
.daily-focus-banner.is-collapsed { max-height: 56px; }
.daily-focus-banner.is-expanded { max-height: 600px; /* generous ceiling */ }
```

## Persistence

Use `localStorage` with key `klosure:focusBanner:lastCollapsedDate`. Stores `'YYYY-MM-DD'`.

- On load, if today's date matches the stored value → start collapsed
- On collapse, write today's date
- On manual expand, do not clear the stored value (collapsing once per day is enough)

## Empty / loading states

- Loading: same skeleton as today, full size
- No active deals: hide entirely (already handled)
- Errored: hide entirely (already handled)
